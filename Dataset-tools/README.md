Dataset-tools
================


## data_augmentation
To flip the images only 

```lua
th data_augmentation -p '/your/path/to/folder' -j 0 
```

To translate and flip 
```lua
th data_augmentation -p '/your/path/to/folder' -j 15
```

To just transtale 
```lua
th data_augmentation -p '/your/path/to/folder' -f
```
